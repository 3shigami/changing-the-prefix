Это приложение позволяет вам управлять своей библиотекой книг. Вы можете добавлять, просматривать, искать и удалять книги.

Функциональные возможности:

Добавление книги:
	Выберите пункт 1 в меню.
	Введите название книги.
	Введите имя автора.
	Введите описание книги.
	Введите жанр книги.
Просмотр списка всех книг:
	Выберите пункт 2 в меню.
	Выводит список всех книг в базе данных.
	Для каждой книги отображается ID, название и автор.
Поиск книг:
	Выберите пункт 3 в меню.
	Введите ключевое слово.
	Приложение найдет все книги, в названии или имени автора которых встречается это слово.
Удаление книги:
	Выберите пункт 4 в меню.
	Введите ID книги, которую хотите удалить.
	Книга будет удалена из базы данных.
	Начало работы:

Запустите приложение:
	Скачайте язык программирования питон с сайта https://www.python.org/
	В командной строке перейдите в папку с приложением.
	Введите в консоль cd "Путь к папке с файлом"
	Введите в консоль python main.py.
